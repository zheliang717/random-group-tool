# 随机分组工具

一个简单易用的在线随机分组工具，支持自定义组数、手动输入姓名、从 Excel 导入人名，并能将分组结果导出为 Excel 表格。

👉 [在线使用](https://zheliang717.github.io/random-group-tool/group-tool.html)

## ✨ 功能特点

- **自定义分组数**：根据需求设定想要分成几组
- **手动输入姓名**：直接在文本框中每行输入一个姓名
- **Excel 导入**：支持上传 `.xlsx` / `.xls` / `.csv` 文件，自动提取第一列作为姓名
- **均衡随机分组**：每次随机打乱，保证各组人数尽可能平均（相差不超过 1 人）
- **结果导出**：一键将分组结果导出为 Excel 表格，方便保存或进一步处理
- **两种模式**：未输入姓名时，自动根据“总人数”生成编号 1~N

## 🚀 使用方法

1. 打开[在线工具](https://zheliang717.github.io/random-group-tool/group-tool.html)
2. 设置分组数（默认 4 组）
3. 输入姓名（二选一）：
   - 手动在文本框中每行输入一个姓名
   - 点击“选择 Excel 文件”导入姓名（支持追加或替换）
4. 点击 **“随机均衡分组”** 即可生成分组结果
5. 如需保存，点击 **“导出结果为 Excel”** 下载表格

## 📦 本地运行

如果您想在自己的电脑上运行，只需下载 `group-tool.html` 文件，用浏览器打开即可，无需安装任何依赖。
